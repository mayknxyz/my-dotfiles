## Useful Commands

```bash
# Find what commands a package provides
pacman -Ql <package> | grep /usr/bin/

# Find which package owns a command
pacman -Qo $(which <command>)

# Search for packages
pacman -Ss <keyword>

# Get package info
pacman -Qi <package>
```
